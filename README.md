# Api Design Practical

Course to learn about API Design

## Slide
Path: `slide/pages/slide.html`
or run on slidev

### Install Slidev

**Go to path `slide`**
```bash
npm install
```

### Run Slidev
**On command line**

```bash
npm run dev
```
**On Browser**
Open `slide/pages/slide.html`

## Code

Path: `src/api`

### Run Code

**On command line**

```bash
go mod tidy
```

**On Command Line**

```bash
cd src/api/cmd/server
go run main.go
```

**Result:**

```bash
2025/01/14 09:37:26 Server is running at http://localhost:4000
```
